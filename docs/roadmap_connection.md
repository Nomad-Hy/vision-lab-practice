아직 포트폴리오로 보여줄 수준은 아니지만,  
나중에 아래 프로젝트들의 기초가 됩니다.

2. 저조도/반사/블러 강인성 실험
3. 고해상도 이미지 품질 분석
4. OpenCV vs AI vs 3D Vision 비교 프로젝트
