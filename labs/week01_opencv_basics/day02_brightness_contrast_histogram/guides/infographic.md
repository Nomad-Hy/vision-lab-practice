# Infographic Guide

이 폴더의 `assets/`에는 Day 02 개념을 설명하기 위한 도식이 들어 있다.

## 포함된 파일

```text
assets/day02_pipeline.svg
assets/brightness_contrast_histogram_summary.svg
```

## 사용 방법

README나 발표 자료에서 아래 흐름을 설명할 때 사용한다.

```text
입력 이미지
→ 흑백 변환
→ 밝기/대비 조정
→ 히스토그램 분석
→ 평활화
→ 실패 케이스 기록
```

README에 넣을 때는 다음처럼 사용할 수 있다.
